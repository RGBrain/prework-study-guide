HTML

  • The head element contains information ('metadata') about the webpage
  • The body element represents visible content shown to the user
  • HTML provides the structure and content for the page
  • Intelligent use of sections and tags helps with SEO and accessibility

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS can be applied to many different things (elements, classes, ids etc)
  • Using an external stylesheet is generally best paractice

Git

  • Typing 'git status' in the CLI checks what branch we are on
  • Create new branches from the most recent version of the main file
  • Commit regularly, after each discrete completed change
  • Use different branches for separate, unrelated changes
  • Typing 'git checkout -b *name of branch*' creates a new branch and switches to it
  • Delete feature branches after merge

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • You can comment multiline comment with /* comment here */
  • For single line comments, the format is // comment here
