Prework Study Guide

HTML

  • The head element contains information ('metadata') about the webpage.
  • The body element represents visible content shown to the user.
  • HTML provides the structure for a page, the text and images etc
  • Intelligent use of sections and tags helps with SEO and accessability

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • elements can be styled, or classes, for instance .card
  • Using an external stylesheet is generally best paractice.

Git

  • git status checks what branch we are on.
  • create new branches from the most recent version of the main file.
  • commit regularly, after each discrete completed change.
  • use different branches for separate, unrelated changes.
  • git checkout -b branch-name: creates a new branch and switches to it
  • delete feature branches after merge
